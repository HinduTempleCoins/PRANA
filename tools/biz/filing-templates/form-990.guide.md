# Guide — 990 Series (Exempt-Organization Annual Return)

> **General information only — NOT legal or tax advice.** Which variant applies and how to
> complete it depend on your organization's facts. Verify with the IRS and consult a
> licensed professional. You file with the IRS yourself.

## What it is
The 990 series is the IRS annual information return for **tax-exempt organizations**.
Variants include **990, 990-EZ, 990-N (e-Postcard), and 990-PF** (private foundations).

## Which variant
Eligibility generally depends on gross receipts, total assets, and organization type. The
tool collects those figures so **you** can determine which variant you are required to
file (per current IRS thresholds) — it does not decide this for you.

## Why it matters
Filing the correct 990 on time is part of **maintaining** exempt status. Repeated failure
to file can lead to automatic revocation. (This pairs with the compliance calendar, FF2-6.)

## How/where you file
With the IRS, by the method the current instructions specify (several variants are
e-filed). This tool produces a completed return for **you** to submit.
